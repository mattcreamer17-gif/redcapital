# RedCapital · Landing Page

A bespoke single-page landing built for the RedCapital conversation at eMerge Americas 2026 (Startup Spotlight SS92). Designed to feel native to the redcapital.cl visual system: Nunito Sans (100 through 900), warm orange-to-red gradient CTAs, teal success accent, warm neutral grays. The page walks a prospect through the pitch hook (Warren ya decide), the 60/30/10 frame, the VigilOre case (160 hours to under 5 minutes), the Ethics Engine paper, the credentials stack, and the Calendly CTA for Matt Creamer.

Language: Spanish primary with an ES/EN toggle in the header. Copy written to respect the voice rules: no em dashes, no antithesis patterns, no performed questions, no hype words. Pricing and revenue-share terms are stripped in line with landing-page disclosure rules. KPMG UK anchored as Big Four, NLP Logix referenced with the mandatory since-2011 / 150-data-scientists bio, Ethics Engine includes the one-sentence psychometric description and both live links.

Intended URL path: /for/redcapital
